```

│
└─ User.js Schema 新增欄位（如有必要）
   survivalMinutes: { type: Number, default: 0 }  // 本次生存累計分鐘
```

**驗收標準：** 長時間掛機 decay 遞減、離線恢復正常、集體負載有效

### 死亡狀態道具限制（v1.10.2 一併實作）

```
目前問題：
  健康度歸零後仍可隨意購買/使用任何道具，
  包含「隨身碟」這種會扣血的東西。
  死機狀態下還能繼續賭博，賭到壞結果再買發電機復活，形成無限迴圈。

v2.0.0 新規則：
  健康度 ≤ 0（死機）時：

  黑市商城限制：
  ├── 可購買：備用發電機 (Generator) only
  ├── 可購買：其他全部鎖定（顯示「伺服器已死機」提示）
  └── 替代方案：免費廣告復活（仍可使用）

  背包使用限制：
  ├── 可使用：備用發電機（強制重啟至 20% HP）
  ├── 不可使用：液態氮、超頻晶片、防火牆（死機開 buff 無意義）
  ├── 不可使用：量子散熱塔（已內建檢查）
  ├── 不可使用：神祕隨身碟（禁止賭博）
  └── 不可使用：霓虹燈管（死機裝飾無意義）

  實作方式：
  ├── shopService.buyItem() 開頭檢查 user.health
  ├── shopService.useItem() 開頭檢查 user.health
  └── 僅 itemId === 'generator' 或 item.effect === 'revive' 允許通過
```

**驗收標準：** 購買、使用、效果邏輯正確；舊道具在背包中仍然可用；死亡時只能買/用發電機
